# User Stories

## Story 1

As a Sales Manager,

I want to view monthly sales trends,

So that I can identify revenue decline early.

---

## Story 2

As a Marketing Manager,

I want to identify high-value customers,

So that I can target them with loyalty campaigns.

---

## Story 3

As a CEO,

I want to analyze regional performance,

So that investment decisions can be optimized.

---

## Story 4

As a Finance Manager,

I want to identify low-profit products,

So that pricing strategies can be improved.
